Detecting Misinformation using NER, Knowledge Graphs, and GNNs

A fake news detection system that represents articles as nodes in a knowledge graph, connects them to named entities, publishing domains, fact-checking sources, and sentiment labels, then classifies misinformation using graph neural networks rather than treating each article in isolation.

Problem

Most fake news classifiers treat each article as an isolated text classification problem. This project tested a different hypothesis: misinformation leaves relational fingerprints. Articles from the same domains, mentioning the same emotionally charged entities, or sharing similar sentiment patterns can be jointly suspicious in ways that a per-article model may miss.

The question was whether a knowledge graph + GNNcould capture those relationships and outperform per-article models on FakeNewsNet.

Data & Feature Engineering

The final dataset contained 21,407 cleaned articles from FakeNewsNet, combining PolitiFact and GossipCop sources with a roughly 3:1 real-to-fake imbalance. All models used weighted cross-entropy to reduce majority-class collapse.

We compared three NER pipelines to see how entity extraction quality affects the downstream graph: spaCy en_core_web_lg, BERT dslim/bert-base-NER, and RoBERTa Jean-Baptiste/roberta-large-ner-english. Article titles were embedded with all-MiniLM-L6-v2 Sentence Transformers, producing 384-dimensional article node features. VADER sentiment converted each title into a positive, negative, or neutral label.

Knowledge Graph Design

The graph represents articles as the central node type and connects them to source domains, verification sources, named entities, and sentiment labels. Entity co-occurrence edges connect entities mentioned in the same headline, while reverse edges are added for bidirectional message passing.

  • 01
    Article nodes21,407 articles with 384-dimensional MiniLM headline embeddings.
  • 02
    Domain nodes2,324 publishing domains extracted from article URLs.
  • 03
    Entity nodesNamed entities extracted by spaCy, BERT-NER, or RoBERTa-NER.
  • 04
    Sentiment nodesPositive, negative, and neutral VADER labels used as explicit graph structure.
  • 05
    Verification nodesPolitiFact and GossipCop fact-checking sources encoded as graph nodes.

Modeling Approach

We evaluated both homogeneous and heterogeneous graph neural networks. The homogeneous baseline flattened every node and edge type into a single graph and used standard GCNConv layers. The heterogeneous model preserved node and edge types using HeteroConv with SAGEConv, allowing separate message-passing behavior for article-domain, article-entity, article-sentiment, article-verification, and entity-entity relationships.

Each configuration was tuned with Optuna Bayesian optimization. The experimental grid covered 8 configurations (3 NER models × 2 graph types, with sentiment ablations for spaCy), with 50 trials each for 400 total model trainings.

Key Findings

  • 01
    Heterogeneous message passing mattered.Preserving node and edge types generally outperformed flattening the graph, with the RoBERTa heterogeneous model beating its homogeneous counterpart by roughly 5 percentage points.
  • 02
    Sentiment edges added signal.Removing sentiment from the spaCy heterogeneous graph reduced test accuracy from 83.37% to 80.76%, confirming that headline tone contributed useful relational information.
  • 03
    NER and graph architecture interact.RoBERTa performed best with the heterogeneous graph but worst in the homogeneous setting, showing that entity extraction quality and GNN design must be selected jointly.
  • 04
    Headline-only relational modeling was competitive.The best model reached 84.03% test accuracy using article titles and graph structure, improving on a comparable 73% propagation-based baseline without relying on social propagation data.

What I Would Do Next

Adding social propagation edges from FakeNewsNet's tweet IDs would be the most impactful extension, since fake and real news tend to spread through different user communities. Other directions: full-article embeddings instead of headline-only, testing generalization across PolitiFact and GossipCop separately, and GNNExplainer attribution to see which entities and domains drive each prediction.
